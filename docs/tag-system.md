# Tag System

## 文档目标

本文件定义系统的基础标签。标签是整个推荐系统的最小计算单位。所有测试题的选项只影响标签分数；人生阶段、阅读人格和当前课题都由标签分数进一步计算得出。

系统共设 12 个标签：growth、achievement、meaning、self_awareness、aesthetics、relationship、society、future、creation、stability、wealth、freedom。

## 标签总表

| 标签 | 中文名 | 核心含义 | 典型问题 |
|---|---|---|---|
| growth | 成长驱动 | 学习、效率、认知提升、能力增强 | 我如何变得更强？ |
| achievement | 成就驱动 | 成功、竞争、职业发展、影响力 | 我如何取得更高成就？ |
| meaning | 意义探索 | 哲学、价值观、人生意义 | 我为什么而活？ |
| self_awareness | 自我觉察 | 心理、情绪、自我理解 | 我为什么会这样？ |
| aesthetics | 审美体验 | 艺术、文化、生活方式、感受力 | 怎样活得更有质感？ |
| relationship | 关系连接 | 亲密关系、家庭、社交、沟通 | 我如何建立更好的关系？ |
| society | 社会观察 | 商业、历史、社会、城市、制度 | 世界如何运转？ |
| future | 未来趋势 | AI、科技、创新、未来变化 | 未来会发生什么？ |
| creation | 创造表达 | 创业、创作、表达、产品 | 我能创造什么？ |
| stability | 稳定建设 | 长期主义、秩序、规划、安全感 | 我如何建立稳定系统？ |
| wealth | 财富认知 | 投资、财务自由、资产配置 | 我如何积累长期财富？ |
| freedom | 自由探索 | 自由、体验、冒险、突破路径 | 我如何摆脱固定路径？ |

## 标签解释

### growth

成长驱动。用户关注学习、效率、知识结构、能力提升和认知升级。高分用户通常相信人生问题可以通过学习、训练和方法论改善。

### achievement

成就驱动。用户关注职业发展、竞争优势、社会认可、影响力和结果。高分用户通常有较强目标感，关心自己是否在向上移动。

### meaning

意义探索。用户关注价值观、人生意义、幸福、存在感和选择的理由。高分用户不满足于“更成功”，更关心“为什么要这样生活”。

### self_awareness

自我觉察。用户关注心理状态、情绪模式、关系中的自己和内在成长。高分用户倾向于向内看，试图理解自己为何成为现在这样。

### aesthetics

审美体验。用户关注艺术、设计、文化、生活方式、氛围和感受力。高分用户更容易被“生活质感”和“精神审美”吸引。

### relationship

关系连接。用户关注亲密关系、家庭、沟通、陪伴和人与人之间的连接。高分用户会把关系质量视为人生质量的重要组成部分。

### society

社会观察。用户关注商业、历史、经济、制度、城市和社会结构。高分用户倾向于从更大的系统理解个人处境。

### future

未来趋势。用户关注 AI、科技、产业变化、未来机会和时代转折。高分用户容易被新技术、新趋势和未来叙事吸引。

### creation

创造表达。用户关注创业、创作、表达、产品、作品和个人项目。高分用户不满足于消费内容，更希望创造属于自己的东西。

### stability

稳定建设。用户关注秩序、规划、长期主义、安全感、家庭和人生系统。高分用户希望把生活经营成一个可靠结构。

### wealth

财富认知。用户关注投资、财务自由、资产配置、资源积累和长期收益。高分用户会把财富视为人生选择权的基础。

### freedom

自由探索。用户关注自由、体验、冒险、转型和摆脱单一路径。高分用户讨厌被安排，渴望更多可能性。

## 标签计算原则

每道题每个选项可以给 1 到 3 个标签加分。第一版建议每个选项总分为 3 分，例如主标签 +2，副标签 +1。所有标签初始分为 0。用户完成问卷后，系统统计 12 个标签总分，并提取 top1、top2、top3 作为后续阶段、人格和当前课题计算的主要输入。
